HTML & CSS Cheatsheet

In this Cheatsheet you are going to find some explanation for HTML & CSS. Feel free to use it.

By Marcelo Martinez

HTML Content

Tag Name Description
<h1-h6> Header Header 1-6 varies size of the text
<p> Paragraph Defines a paragraph of text
<a> Anchor Defines a hyperlink, used to link from one page to another
<div> Division Defines a division or section in an HTML document
<span> Span Defines a section in a document, used for grouping inline-elements
<ul> & <ol> List <ul> defines an unordered list, <ol> defines an ordered list
<li> List Item Defines a list item
<img> Image Embeds an image in an HTML page
<table> Table Defines a table
<tr> Table Row Defines a row in a table
<td> Table Data Defines a cell in a table
<th> Table Header Defines a header cell in a table

CSS Content

Property Name Description
color Text Color Sets the color of the text
background-color Background Color Sets the background color of an element
font-size Font Size Sets the size of the text
margin Margin Sets the margin area around an element
padding Padding Sets the padding area around an element
border Border Sets the border around an element
display Display Specifies the display behavior (block, inline, none, etc.) of an element
position Position Specifies the type of positioning method used for an element (static, relative, absolute, fixed, sticky)
flex Flex Specifies how a flex item will grow or shrink to fit the space available in its flex container
grid Grid Specifies the layout of elements in a grid container
align-items Align Items Aligns flex items along the cross axis
justify-content Justify Content Aligns flex items along the main axis
